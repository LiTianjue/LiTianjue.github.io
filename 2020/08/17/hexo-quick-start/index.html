hexo quick start

1. 本地环境

  • git
  • nodejs

2. 测试运行

  • 打开浏览器查看效果,默认端口4000

3. 新建github 项目

推送刚才新建的目录文件，注意检查.gitignore文件，不要推送public文件夹

4. 将 TravisCI 添加到GitHub中

maketplace中添加 TravisCI
在GitHub中设置CI可以访问代码仓库
GitHub中新建一个Token，前往CI，添加新的环境变量
在项目目录下构建一个CI脚本.travis.yml 
推送这个脚本到代码仓库，则CI开始自动构建

5. 设置部署分支为gh-pages

6. 测试远程访问

https://<用户名>.github.io
